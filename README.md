  * 게시판 조회: /api/board
- 인증 사용자용 API
  * 내 정보 조회: /api/users/me

  * 게시판 좋아요: /api/board/{board_id}/like
  * 게시판 좋아요 기록확인: /api/board/{board_id}/hisotry
